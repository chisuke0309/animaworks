## 重大な設計変更（このセッション）

### ❗ X投稿フローを「承認制」→「自動投稿＋事後通知」に変更

**変更理由**: 承認フローが複数の障害を引き起こしていた
- Telegram承認依頼が3通来る（重複送信）
- LLMが call_human で何度も報告・承認依頼を送る
- 承認タイミング（OK返信）と投稿実行タイミングのずれで3AM投稿事故
- ユーザー判断: 「内容がまずいものはなかった。投稿したらその記事をTelegramに投げるように」

**新しいフロー:**
```
21:00 LLM → kuroにmorning枠制作依頼 → x_post_save_pending(slot="morning") → 終了
08:00 command → x_post_execute_pending(slot="morning") → 投稿 → Telegram自動通知

09:00 LLM → kuroにevening枠制作依頼 → x_post_save_pending(slot="evening") → 終了
17:00 command → x_post_execute_pending(slot="evening") → 投稿 → Telegram自動通知
```

**変更したファイル:**
