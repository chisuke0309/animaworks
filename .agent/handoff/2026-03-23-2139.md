## 技術メモ

### Animaモデル設定の優先順位
```
status.json > config.json
~/.animaworks/animas/<name>/status.json の "model" キーが最優先
```

### サーバー強制停止手順
```bash
pkill -f "supervisor.runner"
launchctl unload ~/Library/LaunchAgents/com.animaworks.server.plist
launchctl load ~/Library/LaunchAgents/com.animaworks.server.plist
```
