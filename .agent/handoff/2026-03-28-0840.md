### 2. パイプライン分離バグ修正（未コミット）
- **問題**: 同時刻の heartbeat と cron が別パイプラインとして表示される
- **原因**: `cron_tick()` が pipeline_id を即座に判定 → heartbeat 未完了なら空のまま別パイプライン生成
- **修正**: `_resolve_pipeline_id()` メソッド新設。heartbeat 実行中なら最大90秒待機してから pipeline_id を取得。判定を `_run_cron_task()` 内に移動

### 3. X投稿マークダウン記法除去（未コミット）
- **問題**: kuro が生成する投稿テキストに `**太字**`, `# 見出し` 等のマークダウン記法が混入 → Xでは効かずそのまま表示
- **修正**: `_strip_markdown()` 関数を追加。太字/イタリック/取消線/インラインコード/見出し/箇条書きマーカーを除去
- **適用箇所**: `post_tweet()` と `save_pending_post()` の両方（保存時点 + 投稿時点の二重サニタイズ）
